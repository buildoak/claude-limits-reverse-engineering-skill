# .private/

Personal calibration data, analysis scripts, and scratch work.
This directory is gitignored -- nothing here is pushed to remote.

## What goes here

- Personal calibration analysis scripts
- Dashboard screenshots
- Usage observations with hardcoded timestamps/paths
- Scratch fitting notebooks
- Anything machine-specific or data-specific

## Calibration store

The CLI calibration data lives at `~/.config/token-track/calibrations.json`, not here.
This folder is for ad-hoc analysis that references that data.
